# ANDROID CSV

library for exporting data to csv

## How to use

Download via gradle : 

```gradle
compile 'com.github.abdymm:androidcsv:0.1.5'
```

or Maven:
```xml
<dependency>
  <groupId>com.github.abdymm</groupId>
  <artifactId>androidcsv</artifactId>
  <version>0.1.5</version>
  <type>pom</type>
</dependency>
```
